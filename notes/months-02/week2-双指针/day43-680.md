## #680 验证回文串2

**【题目描述】**
给你一个字符串 s，最多 可以从中删除一个字符。

请你判断 s 是否能成为回文字符串：如果能，返回 true ；否则，返回 false 。

**【关键词】**
双指针

**【核心技巧】**
1. 双指针从头尾向中间扫描：i 从前往后，j 从后往前
2. 字符匹配时继续移动：s[i] == s[j] 则 i++, j--
3. 遇到不匹配时决策：
- 此时子串 s[i:j+1] 不是回文
- 题目允许删除一个字符，有两种选择：
  - a. 删除左边字符：检查 s[i+1:j+1] 是否回文
  - b. 删除右边字符：检查 s[i:j] 是否回文
- 用 or 连接，只要一种成功即可
4. 扫完全部字符都匹配：说明本身就是回文，直接返回 True
